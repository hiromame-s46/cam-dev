# Boardly

板書写真から黒板を自動検出し、歪みを補正してPNGまたはPDFで保存できる学生向けウェブアプリです。

## 特長

- 黒板らしい暗色領域をブラウザ内で自動検出
- 四隅をドラッグして切り抜き範囲を微調整
- 最大2,800pxの高解像度で歪み・コントラストを補正
- PNG画像または1ページPDFとして保存
- 写真を外部サーバーへ送信しないローカル処理
- スマートフォンのカメラ入力とドラッグ＆ドロップに対応

## 開発

Node.js 22.13.0以降を使用してください。

```bash
npm install
npm run dev
```

品質チェック:

```bash
npm test
npm run lint
```
